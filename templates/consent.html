Participant Information

This study investigates how different password policies affect usability and security.

During this study, the following data will be collected:

  • Password length
  • Time taken to create a password
  • Number of attempts
  • Password Strength Score

No personally identifiable information will be collected.

No passwords will be collected or stored in any form.

Please do NOT use any real passwords.

By clicking "Accept", you consent to participate in this study.
